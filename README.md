# BlogdeCaf-
Proyecto personal de frontend: blog de café con artículos y cursos.

## Descripción
Blog sobre café creado con **HTML y CSS puros**, con un diseño **responsive** que se adapta a móviles, tablets y escritorio.  
Cuenta con **cards interactivas** para cada entrada del blog.

## Tecnologías
- HTML5
- CSS3 (Flexbox, Grid)
- Diseño responsive
- Animaciones hover con CSS

## Características
- Sección de artículos con imágenes y descripciones.
- Sección de cursos y talleres.
- Estructura semántica y clara.
